# Contributing

Thank you for your interest in contributing to Meta Docs MCP!

## Reporting Issues

- Use [GitHub Issues](https://github.com/tiagoandrepro/meta-docs-mcp/issues) to report bugs or request features.
- Include steps to reproduce the issue, expected behavior, and actual behavior.
- Mention which MCP client you're using (Claude Desktop, Claude Code, Cursor, etc.).

## Suggesting Improvements

Open an issue describing the improvement and why it would be valuable. We welcome suggestions for:

- New documentation sources to index
- Search quality improvements
- Client compatibility fixes
- Documentation corrections

## Code of Conduct

Be respectful and constructive. We're all here to build better tools for developers.
